Raspberry Pi 4 + HQ camera

With a 64-bit quad-core ARM processor @1.5GHz, up to 8GB of RAM, Gigabit Ethernet and passive cooling, the Raspberry Pi 4 model B is ideally suited for embedded vision applications.

The HQ camera comes with a 12.3 megapixel Sony IMX477 sensor, adjustable back focus and support for C- and CS-mount lenses. Excellent image quality and changeable lenses sets the standard for Raspberry Pi cameras.

VisionAppster Engine

The Raspberry Pi Machine Vision Kit comes with the VisionAppster engine preinstalled. A large set of built-in vision tools for gauging, code reading and more is included for free. All components available for download in the VisionAppster store also work on the Raspberry Pi. Communication is easy with HTTP, and open extension interfaces let you to add custom functionality using Python, JavaScript, C or C++.
