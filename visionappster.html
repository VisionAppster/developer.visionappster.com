Demo 1 - Binarize


1. Click "File" menu and click "Open".
2. Double-click "CookBookBinarize" to load the project.
3. Click the button in the bottom bar to capture next image.
4. Adjust slider in "Binarize" parameter to change its value.
5. Click "Binarize" tool on the flowchart to see input and output values.
6. Drag "Invert" and drop it on the background to create a parameter editor.
7. Click the play button in the bottom bar to activate continuous image capture mode.

Demo 2 - Detect Blobs


1. Click "File" menu and choose "Open".
2. Open "2. CookBookDetectBlobs".
3. Click the button to change image.
4. Click "Image Source" on the flowchart to display input and output parameters.
5. Drag and drop "image" output on top of the "Detect Blobs:Label" window.
6. Click the layers icon on the left side of the image display to open image layers.
7. Change layer stacking order by dragging the bottom (red) layer up.
8. Change opacity to 50% using the slider.
9. Click the color map button ("red") and select "Light colors".
10. Click the button to change image.

Demo 5 - OpenCV Tools


1. Open "CookBookOpenCVTools".
2. Click button to change image. In the image you will see the output of OpenCv Distance Transform function. The colors mark regions with the shortest distance to each black blob. The line surrounding the centers of the blobs connects the points given by OpenCv Convex Hull function.
3. You will see the design, a window showing the original input image from the Image Source, and the result image which is an overlay of a thresholded and inverted image from Binarize Tool and the output of Distance Transform Tool with "Light Colors" color map.
4. Click on "Layer" symbol on the left side of the result image. The overlays are shown. You can adjust the opacity of the layers in Options section.
5. Click on "Tools" menu at the top of the screen to see all the available tools. Scroll the slider right to find section "OPENCV".
6. Drag "Open Cv Hough Circles" tool on top of Image Source. It auto-connects to the Image output.
7. Click on Hough Circles tool and drag and drop "Circle Image" output onto the canvas.
8. Click the button to change image. Plenty of circles appear in the Circle Image, most of which are false positives.
9. Now click on "Match Sensitivity" parameter of Hough Circles tool and change the value from 50 to 100.
10. Now the false positives disappear and the circles which are white in the original image are shown.

Demo 6 - Image Classifier


1. Click "File" menu and click "Open".
2. Open "CookBookImageClassifier".
3. Click the button to change image. 4. Click "Scale Image" tool to see scaling parameters. 5. Click "Image to Tensor". This tool converts a color image into tensor shape 1x3x224x224.
6. Click "Run ONNX Model" tool to see parameters. You'll see "Model file" location from the tool input.
7. Click "Tensor Classification" tool to see parameters. This tool inputs the tensor of confidences and selects those which exceed the give confidence threshold and sort them into descending order.
8. Click "Java Script". This tool inputs the selected indicates and converts them to readable strings.
9. Click the button to change image.

Demo 7 - YOLO Classifier


1. Click File and Open "CookBookYOLOClassifier".
2. Click the button to change image.
3. Click "Scale Image" tool to see scaling parameters. Scaling parameter is 416x416px.
4. Click "Image to Tensor". This tool converts a color image into tensor shape 1x3x416x416.
5. Click "Run ONNX Model" tool to see parameters. You'll see "Model file" location from the tool input.
6. Click "Process Yolo Result". This tool inputs the tensor from ONNX Tool and decodes the class and location data.
7. Click "Java Script". This tool inputs the selected indicates and converts them to readable strings.
8. Click "Java Script" Code parameter to open jacvascript code. After this close the Javascript code dialog.
9. Click "Blur Regions". This tool blurs the bounding boxes of the found objects.
10. Drag the Process Yolo Result Tool "Frame" output parameter on top of the image to make a widget.
11. Click the button to change image.

Demo 8 - Caliper


1. Open "CookBookCaliper".
2. Click the button to change image. In the "Flowchart" You will see
A. the design,
B. a window showing the original input image,
C. yellow widget which shows the position of the caliper,
D. graph of samples of the profile on the caliper line,
E. frame matrix which converts pixels to millimeters.
3. Click on Caliper tool to see the input parameters.
The easiest way to set start and end points of the caliper line is to drag and drop either one on top of the image. The width of the caliper can also be set with the widget by clicking on the yellow circle at the end of the widget and dragging sideways. Setting the correct width is important as it reduces noise on the caliper line. The gauge is measured by measuring the width of the dark gap in the opening of the spanner. Hence, parameter "Gap Type" is set as "Dark Gap".
